# social-mark real time app

## Project setup
```
npm install
```
### Run json-server

```
json-server db.json --watch
```

### Execute websocket server ./server

```
yarn start 
```
### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
npm run lint
```
